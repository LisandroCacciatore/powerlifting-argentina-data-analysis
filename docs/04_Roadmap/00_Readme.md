# Roadmap del proyecto

Este roadmap describe la dirección general del proyecto.
No es un plan cerrado.

## Próximos pasos

- Exploración sistemática del dataset en BigQuery
- Documentación de problemas de calidad de datos
- Queries analíticas más estructuradas

## Evolución futura

- Visualizaciones más integradas
- Análisis aplicado a decisiones de entrenamiento
- Investigación de uso de Vertex AI para asistencia y feedback

El foco seguirá siendo:
criterio, proceso y decisiones bien fundamentadas.
