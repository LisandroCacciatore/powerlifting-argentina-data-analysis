# Slides – Charla: Análisis de datos aplicado al deporte

Esta carpeta contiene el material visual utilizado para una charla introductoria
(10–15 minutos) sobre análisis de datos aplicado al deporte.

Las slides están pensadas como **soporte narrativo**,
no como material técnico exhaustivo.

La explicación completa se da de forma oral
y se complementa con el PDF descargable del proyecto.

---

## Objetivo de las slides

- Introducir el valor del análisis de datos en contextos deportivos reales  
- Conectar experiencia práctica con pensamiento analítico  
- Mostrar que los datos ayudan a **tomar decisiones**, no solo a hacer gráficos  

El foco está en generar comprensión,
no en mostrar herramientas.

---

## Contenido

- Historia de inicio (gimnasio, clases pequeñas, aprendizaje)  
- Problema central: entrenar sin métricas claras  
- Introducción al dataset de OpenPowerlifting  
- Ejemplos simples de análisis  
- Cierre: criterio, proceso y preguntas correctas  

Las slides **no contienen código**
ni detalles técnicos profundos.

---

## Relación con el proyecto

Las slides funcionan como puerta de entrada al proyecto.

- Introducen el problema y el enfoque  
- El PDF profundiza y ordena el contenido  
- El repositorio documenta el proceso técnico y las decisiones  

Cada pieza cumple un rol distinto,
pero todas cuentan la misma historia.
