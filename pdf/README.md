# PDF – Material descargable

Esta carpeta contiene el material en formato PDF que acompaña la charla
y el repositorio del proyecto.

El PDF está pensado como un documento autónomo,
que se puede leer sin recorrer el código ni el repo completo.

---

## Enfoque del material

El PDF funciona como:

- material de lectura posterior  
- guía paso a paso del enfoque  
- documento compartible (portfolio / formación)  

No busca enseñar programación,
sino explicar cómo pensar un problema de análisis de datos aplicado al deporte.

---

## Alcance del PDF

Duración estimada: 4 a 6 páginas.

Incluye:

- Contexto del proyecto  
- Explicación del dataset  
- Enfoque de análisis  
- Ejemplos de gráficos  
- Conclusiones y aprendizajes  

No requiere conocimientos avanzados de programación
ni experiencia previa en análisis de datos.

---

## Relación con el repositorio

El PDF resume y comunica el enfoque del proyecto,
mientras que el repositorio documenta el proceso completo,
las decisiones y los artefactos técnicos.

Ambos se complementan:
el PDF explica el “qué” y el “por qué”,
el repo muestra el “cómo”.

---

## Público objetivo

- Entrenadores  
- Profesionales del deporte  
- Personas interesadas en datos aplicados  
- Recruiters / hiring managers que evalúan criterio y proceso
